# MineRuster

MineSweeper written in Rust using terminal UI.

### Controls
```console
LMB   Reveal Field
RMB   Mark Field
R     Restart
Q/ESC Quit
```

### Usage
```console
MineRuster

USAGE:
    MineRuster.exe [OPTIONS]

FLAGS:
        --help    Prints help information

OPTIONS:
    -d, --difficulty <DIFF>    Sets game difficulty
    -h, --height <HEIGHT>      Sets board height
    -w, --width <WIDTH>        Sets board width
```

### Compile
```console
cargo build
```
